
#### 时间复杂度

平均时间复杂度O(nlogn)最坏情况下为O(n^2) 在数组已经排序好的情况下出现，可以通过随机化或者取中点来避免最差情况

#### c++代码(快排模板）

```c++
void quick_sort(int q[], int l, int r){
    //递归终止情况
    if(l >= r) return ;
    //1. 分成子问题 
   int i = l -1, j = r + 1, x = q[l + r >> 1];
   while(i < j ){
     do i ++; while(q[i] < x);
     do j --; while((q[i] > x);
     if(x < j ) swap(q[i], q[j]);
   }
   //2.递归处理子问题
   quick_sort(q, l, j);
   quick_sort(q, j + 1, r);
   //3. 子问题合并，快排不需要这一步操作，归并排序需要且是核心

}
```
