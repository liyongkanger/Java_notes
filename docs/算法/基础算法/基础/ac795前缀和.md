### ac795前缀和

### 题目描述

输入一个长度为n的整数序列。

接下来再输入m个询问，每个询问输入一对l, r。

对于每个询问，输出原序列中从第l个数到第r个数的和。

#### 输入

```
5 3
2 1 3 6 4
1 2
1 3
2 4
```
#### 输出

```
3
6
10
```

----------
#### 思路
1. 一维前缀和初始化`s[i] = s[i-1] + q[i]`
2. 求`l,r`区间内的前缀和`s[r] - s[l-1]`

```c++
#include<iostream>

using namespace std;
const int N = 100010;
int n,m;
int q[N], s[N];
int main(){
    
    cin >> n >> m;
    
    for(int i = 1; i <= n; i ++) scanf("%d", &q[i]);
    //前缀和初始化
    for(int i = 1; i <= n; i ++) s[i] = s[i - 1] + q[i];
    
    while(m --){
        int l, r;
        cin >> l >> r;
        //求前缀和
        printf("%d\n", s[r] - s[l - 1]);
    }
}
```
