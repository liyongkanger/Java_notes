### lc12正数罗马数字

### 题目描述

```
字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
```

#### 样例

```
输入: 58
输出: "LVIII"
解释: L = 50, V = 5, III = 3.
```

----------

**思路**
1. 将 个 十 百 千 都表示出来找规律
2. 特殊数字表示出来
3. 循环遍历 num - 高位值 拼接答案

```c++
class Solution {
public:
    string intToRoman(int num) {
        //将 个 十 百 千 都表示出来找规律
       int values[] = {
       1000, 
       900, 500, 400, 100,
       90, 50, 40, 10,
       9, 5, 4, 1};
       string reps[] = {
         "M",
         "CM", "D", "CD", "C",
         "XC", "L", "XL", "X",
         "IX", "V", "IV", "I"};

       string res;
       //从小到大循环 从高位到低位循环
       for(int i = 0; i < 13; i ++){
           //找出高位的值
           while(num >= values[i]){
               //减去高位的值
               num -= values[i];
               //答案拼接
               res += reps[i];
           }
       }
               return res;
    }
}
```
