### ac93递归实现组合型枚举

### 题目描述

从 1~n 这 n 个整数中随机选出 m 个，输出所有可能的选择方案


#### 样例
#### 输入
```
5 3
```
#### 输出
----------

```c++
#include<iostream>

using namespace std;

int n, m;

void dfs(int u, int sum , int state){
    if(sum + n - u < m) return ;
    
    if(sum == m){
        for(int i = 0; i < n; i++){
            if(state >> i & 1){
                cout << i + 1 << ' ';
            }
        }
        cout << endl;
        return ;
    }
    
    dfs(u + 1, sum + 1, state | 1 << u);
    dfs(u + 1, sum, state);
}

int main(){
    
    cin >> n >> m;
    dfs(0,0,0);
    
    return 0;
}
```
