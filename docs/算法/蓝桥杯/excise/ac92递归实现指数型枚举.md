### ac92递归实现指数型枚举

### 题目描述
从 1~n 这 n 个整数中随机选取任意多个，输出所有可能的选择方案。

#### 样例
#### 输入
```
3
```
#### 输出
```

3
2
2 3
1
1 3
1 2
1 2 3
```

----------

```c++
#include<iostream>
#include<algorithm>

using namespace std;

const int N = 20;
int n;


void dfs(int u, int state){
    
    if(u == n){
        for(int i = 0; i < n; i ++)
         //查看第i位上是否为1
           if(state >> i & 1)
               cout << i + 1 << " ";
            
       cout << endl;
       
       return ;
    }
    //递归下一次不选
    dfs(u+1, state);
    //选
    dfs(u + 1, state | (1 << u));
    
}
int main(){
    
    cin >> n;
    
    dfs(0, 0);
    
    
    return 0;
}
```
